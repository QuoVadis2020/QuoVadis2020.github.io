函数包含4部分：返回类型，函数名，形参列表，函数体。

编译器

g++：

  • 编译：g++ --std=c++11 main.cpp -o main
  • 运行：./main
  • 获取状态：
    • windows: echo %ERRORLEVEL%
    • UNIX: echo $?
  • 编译多个文件：g++ ch01.cpp Sales_item.cc -o main

IO

  • istream ostream 输入输出流：一个流是一个字符序列。
  • 头文件：使用标准库的设施都应用#include来包含相关头文件，类的类型一般存储在头文件中，标准库的头文件使用<>，非标准库的头文件使用""。申明写在.h文件，定义实现写在.cpp文件。。#include <iostream>
  • endl操纵符：结束当前行，并将设备关联的缓冲区中的内容刷到设备中。若不手动刷新，则需等到缓冲区满才会自动刷新。
  • » « 输入输出运算符：返回的结果都是左操作数，也就是输入流和输出流本身。
  • 命名空间（namespace）：避免名字冲突，标准库定义的所有名字都在std中。也可以使用::作用域运算符在左侧的作用于中寻找右侧那个名字，如std::cout。

注释

  • 单行注释 ：//
  • 多行注释 ：/* */

循环

  • while语句 条件（condition）为假时结束

  • for语句

    循环头：

    • 一个初始化语句（init-statement）
    • 一个循环条件（condition）
    • 一个表达式（expression）

  • i++和++i：i++在语句执行后对i自增，++i在语句执行前对i自增。另外，从运行速度来讲，由于i++返回i的值，而++i返回i的引用，所以对于类类型，++i更快。

类

  • 包含来自标准库的头文件时应用尖括号<>，不属于标准库的头文件用双引号""
  • 文件重定向：大多数系统都可用a.out <infile >outfile来将infile内容作为标准输入，并将标准输出写入到outfile
